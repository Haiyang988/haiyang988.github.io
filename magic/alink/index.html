PageWithMedia

带有媒体内容的页面

概述

PageWithMap组件。包含图面和内容两部分，不包含顶部导航栏。页面导航通过【X】按钮实现。

图示

在真实项目开发中，你可能会需要 Redux 或者 MobX 这样的数据流方案，Ant Design React 作为一个 UI 库，可以和任何 React 生态圈内的数据流方案以及应用框架搭配使用。我们基于 Redux 推出了自己的最佳实践 dva，以及可插拔的企业级应用框架 umi，推荐你在项目中使用。

使用方法

在开始之前，推荐先学习 React 和 ES2015，并正确安装和配置了 Node.js v8 或以上。 官方指南假设你已了解关于 HTML、CSS 和 JavaScript 的中级知识，并且已经完全掌握了 React 全家桶的正确开发方式。如果你刚开始学习前端或者 React，将 UI 框架作为你的第一步可能不是最好的主意。

API

我们在现阶段不严格的借用了原子化设计（Atomic Design）中的一些理念。这个理念由 Brad Frost 于 2013 年提出，旨在通过构建可复用的代码片段（组件）来降低开发成本。这个理念将组件按照原子、分子、有机体、模版和页面划分，原子可构成分子，分子构成有有机体，有机体构成模版 … 以此类推。大概就是这个意思。该兴趣的同学可以去他的网站在开始之前，推荐先学习 React 和 ES2015，并正确安装和配置了 Node.js v8 或以上。 官方指南假设你已了解关于 HTML、CSS 和 JavaScript 的中级知识，并且已经完全掌握了 React 全家桶的正确开发方式。如果你刚开始学习前端或者 React，将 UI 框架作为你的第一步可能不是最好的主意。
